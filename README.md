# phenogamShiny

<!-- badges: start -->

[![Lifecycle:

The goal of phenogamShiny is to model densities of species at a given
site using cumulative degree days.

## Installation

You can install the development version of phenogamShiny like so:

``` r
remotes::install_github("https://github.com/ecodata-technology/phenogamShiny")
```

## Example

This is a basic example which shows you how to run the app:

``` r
library(phenogamShiny)
run_app()
```
